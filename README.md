## RUNNING

-   Install all dependencies:
    `yarn`

-   Set up env: create `.env` file by copying data in `.env.example`

-   Run docker:
    `yarn:docker:up`

-   Run user services:
    `yarn dev:user`

-   Run review services:
    `yarn dev:review`

-   Need to down all docker container
    `yarn docker:down`
